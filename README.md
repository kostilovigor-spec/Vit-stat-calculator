# Vitruvium — калькулятор закупа

Одностраничный калькулятор для создания персонажа по правилам закупа Vitruvium. Позволяет вручную задать базовые уровни, а затем тратить опыт на повышение характеристик, черт и способностей.

## Возможности
- Ввод уровня или количества опыта (формула: **30 + уровень × 6**).
- Ручной ввод базовых характеристик/черт/способностей без стоимости.
- Закуп через кнопки **+ / -** с автоматическим учетом стоимости и остатка опыта.
- Ограничение уровней 0–6 (для характеристик минимум 1).
- Итоговая сводка расходов и остатка опыта с подсветкой при уходе в минус.
- Черты и способности объединены в один раздел с выбором типа в строке.

## Как пользоваться
1. Откройте `index.html` в браузере.
2. В правой колонке задайте уровень и/или опыт.
   - Если включена галочка **XP по уровню**, опыт считается автоматически.
3. В левом блоке задайте базовые значения характеристик (без стоимости).
4. В разделе **Черты и способности** добавляйте строки и выбирайте тип, базу и покупку.
5. Следите за **Итогом по опыту** справа.

## Правила расчета (по таблицам)
- **Характеристики** (уровни 1–6): стоимость повышения по уровням — 0, 3, 6, 9, 12, 15.
- **Черты/способности** (уровни 1–6): стоимость повышения по уровням — 1, 2, 4, 6, 8, 10.
- **Полная стоимость** считается как сумма всех уровней от базы + 1 до текущего.

## Сброс
Кнопка «Сбросить к 1 уровню»:
- возвращает уровень на 1 и опыт на 36;
- характеристики — база и покупка в 1;
- черты/способности **не удаляются**, их уровни сбрасываются в 0.

## Структура проекта
- `index.html` — разметка.
- `styles.css` — стили.
- `app.js` — логика расчета.

## Настройка
Если нужно изменить лимит уровней или стоимость повышения, редактируйте `MAX_LEVEL` и `costTables` в `app.js`.
